ตั้งค่าครั้งเดียว ทุกคนในทีมใช้ได้เลย
1
สร้าง Firebase Project (ฟรี)
ไปที่ console.firebase.google.com → Add project → ตั้งชื่อ → ปิด Analytics → Create project

จากนั้น: Build → Realtime Database → Create database → เลือก region → Start in test mode
2
คัดลอก Database URL
จะเห็น URL ในรูปแบบ:
https://your-project-default-rtdb.firebaseio.com
3
ตั้งชื่อทีมและ Telegram
ชื่อทีม (แสดงใน Telegram)
Telegram Bot Token (ไม่บังคับ)
Telegram Chat/Group ID (ไม่บังคับ)
4
ชื่อของคุณ
ใช้แสดงในรายงาน
แชร์ URL ไฟล์นี้ + config เดียวกันให้ทีม ทุกคนจะ sync กัน
